# AGENTS.md

이 저장소는 공개 가능한 기술 교류 문서를 보관하는 문서 저장소다.

## 프로젝트 개요

PROJECT: peach-tech-notes
TYPE: 공개 기술 교류 문서 저장소
PURPOSE: PeachSolution 팀이 축적한 공통 기술 문서, 도구 가이드, 운영 메모를 정리

## 핵심 원칙

- 프로젝트 내부 전략, 고객 정보, DB 구조, 운영 경로는 문서에 넣지 않는다.
- `peach-harness`와 역할이 겹치지 않도록 하네스 정본 문서는 이 저장소에 복제하지 않는다.
- 문서는 한국어로 작성한다.
- 폴더와 파일은 번호형 구조를 유지한다.
- 새 문서를 추가할 때는 어느 카테고리에 속하는지 먼저 판단한다.

## 문서 구조

```
peach-tech-notes/

```

## 추가/수정 판단 기준

- 워크플로우/검증 절차 → `1.개발플로우`
- E2E/브라우저 자동화 → `2.E2E`
- Claude Code 사용/스킬/플러그인/마켓플레이스 → `3.클로드코드`
- MCP/외부 도구/문서 변환 도구 → `4.도구연동`
- 외부 방법론/프레임워크 → `5.외부프레임워크`
- 위키/지식 축적/문서화 시스템 → `6.지식관리`

## 제외 대상

- 고객명, 프로젝트 내부 경로, 실제 DB/테이블/컬럼 구조
- 사업 전략, 계약 구조, 수익 구조
- 내부 운영 메모, 사람 평가, 팀 갈등 맥락
