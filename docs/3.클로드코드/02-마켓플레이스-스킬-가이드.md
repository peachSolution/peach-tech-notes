| 스킬명 | 기능 | 사용 예시 |
|:------:|:-----|:----------|

| **pdf** | PDF 텍스트 추출, 생성, 병합/분할, 양식 처리 | `/pdf` |

| **webapp-testing** | Playwright로 로컬 웹앱 테스트 및 디버깅 | `/webapp-testing` |
| **frontend-design** | 프로덕션급 프론트엔드 UI 디자인 | `/frontend-design` |

| **docx** | Word 문서 생성/편집 (추적 변경, 주석) | `/docx` |
| **web-artifacts-builder** | React/Tailwind/shadcn으로 복잡한 웹 아티팩트 빌드 | `/web-artifacts-builder` |

## 2. 주요 스킬 심층 가이드

### 2.1 skill-creator (스킬 생성)

```
┌─────────────────────────────────────────────────────────────────┐
│  ⭐ skill-creator - 새로운 스킬 제작 도구                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  📋 용도                                                        │
│  ├── 새로운 커스텀 스킬 생성                                     │
│  ├── 기존 스킬 개선/수정                                         │
│  └── 스킬 템플릿 제공                                           │
│                                                                 │
│  🔧 사용법                                                      │
│  /skill-creator "스킬 설명"                                     │
│                                                                 │
│  💡 팁                                                          │
│  - 스킬 생성 시 반드시 이 스킬로 시작                             │
│  - 명확한 트리거 조건 정의 필수                                   │
│  - 테스트 후 배포                                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**스킬 생성 워크플로우:**

```
  요구사항 정의 ──▶ /skill-creator ──▶ 스킬 파일 생성 ──▶ 테스트 ──▶ 배포
       │                   │                  │              │
       ▼                   ▼                  ▼              ▼
   "어떤 기능?"      템플릿 선택         .md 파일       동작 확인
```

### 2.2 xlsx (엑셀 파일 생성)

```
┌─────────────────────────────────────────────────────────────────┐
│  📊 xlsx - 스프레드시트 완전 지원                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ✅ 지원 기능                                                   │
│  ├── 생성: 새 스프레드시트 (.xlsx, .xlsm)                        │
│  ├── 읽기: 기존 파일 분석 (.xlsx, .csv, .tsv)                    │
│  ├── 편집: 수식 유지하며 수정                                    │
│  ├── 수식: SUM, VLOOKUP, IF 등 모든 수식                        │
│  ├── 포맷: 셀 서식, 조건부 서식                                  │
│  └── 시각화: 차트 생성                                          │
│                                                                 │
│  📝 사용 예시                                                   │
│  /xlsx "매출 데이터 분석 스프레드시트 생성"                       │
│  /xlsx "이 CSV 파일을 엑셀로 변환하고 피벗 테이블 추가"           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

| 작업 | 명령 예시 |
|:----:|:---------|
| 새 파일 생성 | `/xlsx "월별 매출 보고서 생성"` |
| 데이터 분석 | `/xlsx "이 데이터의 트렌드 분석"` |
| 수식 추가 | `/xlsx "합계와 평균 수식 추가"` |
| 차트 생성 | `/xlsx "막대 차트로 시각화"` |
