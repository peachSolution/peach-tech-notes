# peach-tech-notes

PeachSolution 팀이 축적한 공개 가능한 기술 교류 문서를 모아두는 저장소입니다.

`peach-harness`가 하네스 정본이라면, 이 저장소는 도구 사용법, 운영 메모, E2E 방법론, 외부 프레임워크 정리처럼 기술 공유에 초점을 둡니다.

## 원칙

- 프로젝트 내부 전략 문서는 두지 않습니다.
- 고객/도메인/DB/운영 경로가 드러나는 문서는 두지 않습니다.
- 공개 가능한 일반화 문서만 유지합니다.
- 하네스 정본 문서와 역할이 겹치지 않도록 구분합니다.

## 문서 구조

- `docs/1.개발플로우` — 개발 절차, 검증 전략
- `docs/2.E2E` — 브라우저 테스트, 도구 비교, 시나리오
- `docs/3.클로드코드` — Claude Code 중심 운영 문서
- `docs/4.도구연동` — MCP, 문서 도구, 외부 도구 연동
- `docs/5.외부프레임워크` — 외부 프레임워크/방법론 정리
- `docs/6.지식관리` — 지식 축적, 위키 패턴

## 정본 구분

- 하네스 정본: `peach-harness`
- 기술 교류 문서: `peach-tech-notes`
- 피치솔루션 내부 전략 문서: `peach-ai-dev`
